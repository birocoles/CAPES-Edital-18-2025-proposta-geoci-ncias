# Reflexões e propostas para a ára de Geociências da CAPES

O presente documento foi elaborado no âmbito do Edital CAPES 18/2025, que disciplina o processo de
renovação das coordenações de área da CAPES, especificamente para a área de Geociências.

A proposta está disponível na plataforma Zenodo: [![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.17602928.svg)](https://doi.org/10.5281/zenodo.17602928)

O trabalho pode ser citado no estilo APA da seguinte forma:

Vanderlei C Oliveira Jr. (2025). Reflexões e propostas de atuação frente à área de Geociências (v1.0.0). Zenodo. https://doi.org/10.5281/zenodo.17602928
